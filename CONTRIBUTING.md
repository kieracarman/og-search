# Contributing to OG Search

yo! thanks for wanting to contribute.

## how to contribute

1. fork the repo
2. create a branch (`git checkout -b feature/thing`)
3. make your changes
4. test thoroughly
5. submit a PR

## what we're looking for

- new search engine support
- bug fixes
- performance improvements
- documentation improvements
- UI/UX enhancements

## what we're NOT looking for

- premium features
- tracking/analytics
- ads or monetization

## code style

- use prettier (80 char width)
- lowercase commit messages
- keep it simple

## testing

test on:
- latest firefox stable
- multiple search engines
- with extension on/off

## questions?

open an issue!
